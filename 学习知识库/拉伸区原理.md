# 拉伸区原理

> 出自 try-stretch skill

## 三个区

```
舒适区              拉伸区              恐慌区
 (无聊/停滞)       (成长/投入)        (逃避/焦虑)
    |────────────────|────────────────|
      太简单           刚刚好            太难
```

**舒适区** — 不需要刻意努力，感觉安全。作用是巩固技能、降低认知负荷。**代价：零增长。**

**拉伸区** — 有点难但做得到。信号是「可能会搞砸，但试试不亏」。这是成长发生的地方。

**恐慌区** — 太难了，大脑拒绝。信号是逃避、拖延、想关掉页面。

## 常见误解

- ❌ 「拉伸区 = 一直不舒服」→ 不是，拉伸区有适度的兴奋感
- ❌ 「舒适区是坏的」→ 不是，舒适区用来巩固，有效学习是舒适区→拉伸区→休息→再巩固
- ❌ 「恐慌区 = 挑战」→ 不是，挑战让你兴奋，恐慌让你逃避

## 怎么用

1. 先定位自己在哪一区
2. 如果在舒适区 → 加一个约束/变量
3. 如果在恐慌区 → 退回一步，找一个「知道怎么做但觉得太简单」的任务
4. 如果在拉伸区 → 保持，校准边界
