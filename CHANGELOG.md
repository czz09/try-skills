# Release Notes

## v0.1.0 (2026-07-03)

### 🎉 初始发布

6 个 try- skill 首次发布：

- **try** — 总路由器，帮用户判断卡在哪个环节
- **try-plan** — 目标可行性检查，三档估算（乐观/常规/保守）
- **try-stretch** — 拉伸区定位，舒适区/拉伸区/恐慌区
- **try-method** — 学法匹配，知识/技能/行为/感受四类
- **try-task** — 今日任务分解，从目标到唯一动作
- **try-review** — 巩固策略，主动回忆+间隔重复+复盘

### 适配平台

- Hermes Agent
- Claude Code
